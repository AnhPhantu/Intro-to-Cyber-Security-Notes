# Important Commands for Kali Linux
This notes included important commands and vocab for Kali Linux that would be helpful in my cyber security class. 
## Commands List
-pwd
 * print working directory (use to tell user where they are in directories).
 
-ls, -ls-la, -ls-a, -ls-l
* a command that "list" the current directory (-a shows us the hidden file , -l shows us more information about the file in the directory)

-cd
* a command that change the directory (to move into a certain directory that is chosen).

-mkdir"name"
* a command that make directory with a name

-rmdir"name"
* a command that remove the chosen directory

-man "command"
* a helpful command that give information about the chosen command

-process/programs&
* a command that allow other process and program to run in the background within Kali Linux

-touch "name"
* create a file or update the time stamp

-leafpad "name"
* open the leafpad app for text editing 

-cat "name"
* a command that print what is in the file in the terminal

-firefox "name". file
* a command that open that file in firefox

-wget 
*a command that allows you to download a file if the irl is known

-rm
*a command used to remove file (not directories)

-adduser "name"
* a command that creat a user with password and other information

-useradd "name"
* a command that create a new user without other addtional info

-passwd "username"
* a command to add a password to the user

-sudo
* a command that allow the certain user to perform sensitve action

-whoami
* a command that provide with the acocunt you login as

-userdel "name"
* a command that delete a user

-groupadd
* a command that create a group in the system

-usermod
* a command that is used to make change to user

-groups "name"
*a command that see what group a user is in
